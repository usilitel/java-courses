
Игра "Сапер" v2

описание
-------------------
классы:

class MinerCell - класс описывает состояние ячейки
	свойства:
	boolean isMine // true = мина
	int cellState // текущее состояние ячейки (закрыта, открыта, помечена как мина)
	int cntMinesNear // количество мин вокруг

class MinerField - игровое поле. Содержит методы для отрисовки разных типов ячеек.
	методы:
	drawNumber // рисуем цифру (количество мин вокруг)
	drawBomb // рисуем мину
	drawFlag // рисуем метку
	drawEmptyBlock // рисуем пустой открытый блок
	drawClosedBlock // рисуем закрытый блок

class MinerLogic - логика игры.
	задаются все параметры игры: FIELD_SIZE_X, FIELD_SIZE_Y - размер поля, COUNT_OF_MINES - количество мин, ...

	основные методы:
	ProcessCell(int x, int y, int action) // обрабатываем действие пользователя (открываем/помечаем ячейку)
	GenerateField() // генерируем поле, расставляем мины, заполняем поле MinerCell.CntMinesNear
	CheckGameOver() // проверка на успешное завершение игры

	вложенный класс:
	class MinerFieldPaintable extends MinerField // Расширение класса MinerField (игровое поле для игры "Сапер") для реализации логики отрисовки ячеек. Внутри класса MinerField эту логику не стал реализовывать, чтобы вся логика была внутри класса MinerLogic.
		методы:	
 		paint(Graphics g) // перерисовываем все ячейки
		drawCell(Graphics g, int x, int y) // логика отрисовки отдельной ячейки

Способы расширения функционала (задачи взяты из задания):

Задача: рисовать изображение мин по-другому.
Решение: переопределить метод MinerFieldPaintable.drawBomb

Задача: сделать мины нескольких типов
Решение: 
1) расширить класс MinerCell, добавить свойство mineType.
2) переопределить метод MinerFieldPaintable.drawBomb (рисовать мины в зависимости от поля MinerCell.mineType)
3) в методе MinerLogic.GenerateField заполнять поле MinerCell.mineType

Задача: изменить логику игры так, чтобы взрыв происходил лишь после 3 ошибки
Решение: изменить метод MinerLogic.OpenCell

Задача: изменить логику игры так, чтобы при попадании на мину взрыв происходил с вероятностью 1/3
Решение: изменить метод MinerLogic.OpenCell

Задача: изменить способ ввода информации в программу (использовать клавиатуру вместо мыши)
Решение: сделать обработчик нового события и вызывать из него метод MinerLogic.ProcessCell(int x, int y, int action)
